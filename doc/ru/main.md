# Добро пожаловать в ScL Kit C++

"Проще, чем кажется!"

Комплекс ScL Kit содержит набор модулей С++, предназначенных для упрощения решения некоторых задач с использованием языка программирования C++.

# Модули

* Геометрия
    * spatial - координаты и преобразования между ними
    * geometry - работа с геометрическими формами

* Метапрограммирование
    * meta - набор средств программированиия на этапе компиляции

* Графика
    * gltiny - простая графическая библиотека

* Логирование
    * log - простая система логирования

* Структуры данных
    * feature - средства добавления дополнительных особенностей
	
* Компилятор
    * standard - определения
